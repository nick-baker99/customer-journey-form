# Customer Journey Form - Code Test

Welcome to the Arma Karma code test. Please fork this repository and complete the challenge below.

## Your Task

Build a professional customer journey form application that collects user information, submits it to an API, and displays a success confirmation. Really think about UI/UX, data handling, API routes and most importantly—show us what you can do!

**Time Recommendation:** No longer than 2 hours.

## Requirements

1. **Create a form page** (`app/page.tsx`) that collects customer information
2. **Create an API endpoint** (`/api/submit`) that handles form submissions
3. **Create a success page** (`/success`) that confirms submission
4. **Implement validation** - both client-side and server-side
5. **Add at least 1 dynamic React element using hooks** (e.g., useState, useEffect, or custom hooks)
6. **Style appropriately** - make it professional and user-friendly (Frontend will be a big part of day to day at Arma Karma)

Feel free to design your own customer journey, whether that's inspired by competitors, companies you admire, or completely original ideas. Show us your creativity!

**Optional Challenge:** If you have time within the 2 hours, feel free to add more steps/pages, or integrate a third-party API to enhance the customer journey. Check out [public-apis](https://github.com/public-apis/public-apis) for inspiration, or use any free secure API you prefer.

React is amazing for third-party libraries too. If there's something you love to use, show us what you can do!

You are welcome to add any additional features you see fit.

Once completed, please push your changes and share the repository link.

## Setup

This is a Next.js project with TypeScript and Tailwind CSS pre-configured.

**Install dependencies:**

```bash
npm install
```
